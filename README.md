# QuantumX Foundation Website

A clean, modern website for QuantumX Foundation built with HTML, CSS, and Tailwind CSS.

## Features

- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Modern UI**: Clean, minimalist design inspired by contemporary tech websites
- **Smooth Scrolling**: Smooth navigation between sections
- **Mobile Menu**: Collapsible navigation menu for mobile devices
- **Email Subscription**: Footer email signup form

## Getting Started

Simply open `index.html` in your web browser. The site uses Tailwind CSS via CDN, so no build process is required.

### Local Development

You can use any local server to preview the site:

```bash
# Using Python
python3 -m http.server 8000

# Using Node.js (if you have http-server installed)
npx http-server

# Using PHP
php -S localhost:8000
```

Then visit `http://localhost:8000` in your browser.

## Customization

- **Colors**: Modify Tailwind classes to change the color scheme
- **Fonts**: Currently using Playfair Display (serif) and Inter (sans-serif) from Google Fonts
- **Content**: Edit the HTML directly to update text and sections

## Browser Support

Works on all modern browsers (Chrome, Firefox, Safari, Edge).
